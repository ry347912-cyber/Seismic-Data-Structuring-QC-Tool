Output JSON Schema

Every extracted feature follows this validated structure

{
  "feature_id":            string    // e.g. "F-001"
  "fault_type":            string|null  // normal, reverse, thrust, strike-slip…
  "structure_type":        string|null  // anticline, graben, dome…
  "depth_m":               float|null   // normalised to metres
  "depth_unit":            string|null  // m / ft / km
  "depth_raw":             string|null  // original text excerpt
  "strike":                float|null   // degrees (0–360)
  "dip":                   float|null   // degrees (0–90)
  "dip_direction":         string|null  // N / NE / SE…
  "amplitude":             string|null  // high / low / bright spot…
  "frequency":             string|null  // Hz value or descriptor
  "reflector_continuity":  string|null  // continuous, chaotic, parallel…
  "velocity_ms":           float|null   // m/s
  "age":                   string|null  // Cretaceous, Jurassic…
  "formation":             string|null  // named stratigraphic unit
  "confidence_score":      float        // 0.0 – 1.0
  "missing_fields":        string[]     // required fields absent
  "qc_flags":              string[]     // MISSING_DEPTH, LOW_CONFIDENCE…
  "qc_recommendations":   string[]     // actionable QC advice
}

Quick Start

Get running in under 2 minutes

# 1. Clone
git clone https://github.com/YOUR_USERNAME/seismic-qc.git && cd seismic-qc

# 2. Install dependencies
pip install -r requirements.txt

# 3. Add your Groq API key (free at console.groq.com)
cp .env.example .env && nano .env

# 4a. Run CLI demo (no API key needed)
python main.py --demo

# 4b. Launch Streamlit UI
streamlit run app.py
